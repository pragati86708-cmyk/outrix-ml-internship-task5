# OutriX ML Internship — Task 5

## Diabetes Prediction Model

### Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

### What This Project Does
Predicts whether a person is diabetic or non-diabetic using health features like Glucose, BMI, Age, Blood Pressure and Insulin. Compares Logistic Regression and Random Forest models using accuracy, classification reports, confusion matrices and ROC curves.

### How to Run
Open the .py file in Google Colab and run all cells.
